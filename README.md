# 🏬 MultiFloor BillDesk

A Django-based retail billing and management system designed for multi-floor stores. The application enables administrators and floor supervisors to efficiently manage products, customer billing, inventory, and sales.

---

## Features

- User Authentication
- Role-Based Access Control
- Product Management
- Customer Billing
- Inventory Management
- Sales Tracking
- Responsive User Interface

---

## Tech Stack

- Python
- Django
- HTML5
- CSS3
- JavaScript
- SQL

## Installation

```bash
git clone https://github.com/Suma-s-h/multifloor-billdesk.git

cd multifloor-billdesk

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
